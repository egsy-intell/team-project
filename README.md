# gh-pages

This branch hosts the static site published via GitHub Pages. It does not contain the project's Python code, notebooks, or dependencies — see the `main` branch for that.

## Structure

- `docs/` — site content, served by GitHub Pages
  - `docs/kick-off/ideas/index.html` — kick-off ideas page

## Publishing

Pages are served directly from the files in this branch (configured in the repo's GitHub Pages settings). Add or edit HTML under `docs/` and push to `gh-pages` to update the live site.
